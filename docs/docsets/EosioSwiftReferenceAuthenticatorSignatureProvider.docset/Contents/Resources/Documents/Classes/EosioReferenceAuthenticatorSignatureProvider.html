EosioReferenceAuthenticatorSignatureProvider

public class EosioReferenceAuthenticatorSignatureProvider : EosioSignatureProviderProtocol

Signature provider for EOSIO SDK for Swift that provides selective disclosure and transaction signing using the EOSIO Reference iOS Authenticator App. Conforms to EosioSignatureProviderProtocol.

Requirements for developers.

  • Add manifest.json file.
  • Add URL scheme or Universal Link.
  • Add URL scheme to whitelist.
